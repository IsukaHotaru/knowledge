# 数值存储规范
本文档介绍计算机中的数值存储规范, 包括int类型, float类型(单精度浮点数), double类型(双精度浮点数), INT8, FP16(低精度量化方案)等

一般而言, 精度越低, 存储与计算占用资源越少. 为了尽可能地减少资源占用, 同时又尽可能地减少精度损失, 量化算法应运而生.
